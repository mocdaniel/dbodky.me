# Raspberry Pi 4B

Along with a few [TuringPi RK1 SBCs](./rk1.md) I am running [Raspberry Pi 4Bs](https://www.raspberrypi.com/products/raspberry-pi-4-model-b/) in my homelab. I use them mostly as standalone Docker hosts.

## Specifications

| Spec             | Value   | Notes               |
| ---------------- | ------- | ------------------- |
| **Architecture** | `armv8` |                     |
| **Cores**        | 4       | Cortex-A72 @ 1.8GHz |
| **Memory**       | 8GB     | LPDDR4              |
| **SD Storage**   | varying | varying             |

## Machines

The following machines are present in my homelab:

| Hostname  | IP            | Usage                             |
| --------- | ------------- | --------------------------------- |
| `olympus` | `192.168.1.3` | [Docker Host](../docker/index.md) |
| `rpi-2`   | _not active_  | _not active_                      |
| `rpi-3`   | _not active_  | _not active_                      |
